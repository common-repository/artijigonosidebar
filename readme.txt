== Description ==

Remove sidebar from JigoShop WordPress websites Shop page that uses Artisteer theme

== Installation ==

1. Upload `arti_jigo_nosidebar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
